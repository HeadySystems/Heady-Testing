# Heady Competitive Analysis Prompt Pack

## Best use
Use these prompts with the competitive-analysis skill when you want reusable outputs instead of one-off notes. Fill in the bracketed fields before running them.

## Master prompt
Analyze these competitors for Heady: [competitor list].
Market/category: [market].
Decision this should support: [sales / messaging / content / positioning / fundraising / product].
Primary audience: [audience].
Deliver all of the following:
1. A concise battlecard for each competitor.
2. A side-by-side messaging matrix.
3. A content gap map showing themes, formats, and journey stages.
4. A 2x2 positioning recommendation for Heady.
5. A final section called "How Heady Wins" with the top 5 most defensible moves.
Rules:
- Separate observed facts from interpretation.
- Cite every factual claim with a URL.
- Be brutally honest about competitor strengths.
- Translate findings into action, not just summaries.
- Prefer contrasts that matter to technical buyers, operators, and enterprise decision makers.

## Battlecard prompt
Create a one-page battlecard for [competitor].
Include:
- One-sentence summary
- Their likely buyer
- Their 3 strongest advantages
- Their likely weaknesses or blind spots
- Why Heady wins against them
- 5 trap-setting questions
- 5 objection-handling responses
- When not to position against them
Use a direct enablement tone.

## Messaging matrix prompt
Compare Heady against [competitors].
Build a matrix with these rows:
- Headline
- Subhead / promise
- Primary audience
- Value proposition
- Proof points
- Differentiator claim
- Category framing
- Tone
- Primary CTA
Then answer:
- Which messages sound generic?
- Which claims are well-supported?
- Where is the clearest whitespace for Heady?
- What should Heady stop saying if it sounds too similar?

## Content gap prompt
Analyze content gaps between Heady and [competitors].
Look for:
- Topics competitors cover that Heady does not
- Topics Heady could own because competitors under-explain them
- Missing bottom-of-funnel content
- Missing technical trust content
- Missing founder, vision, or category-creation content
Return a prioritized table with:
- Topic
- Search/user intent
- Competitor currently owning it
- Recommended Heady format
- Priority
- Reason it matters

## Positioning map prompt
Create a positioning map for Heady and [competitors].
First choose the two best axes for this market.
Then:
- Place each competitor conservatively
- Explain why each sits there
- Identify the most under-owned market space
- Recommend the positioning statement Heady should test
- Give 3 headline options and 3 subhead options that fit the recommendation

## Fundraising angle prompt
Analyze [competitors] from an investor and strategic narrative lens.
Focus on:
- Category maturity
- Crowded claims versus true differentiation
- Why Heady could matter as a platform, control layer, or new category entrant
- Which competitor weaknesses suggest a durable opening
Return:
- Market narrative
- Competitive wedge
- Defensibility argument
- Risks to the thesis

## Fast refresh prompt
Update this prior competitive analysis with only materially new information from the last 30-90 days.
For each competitor, identify:
- New product launches
- Pricing or packaging changes
- Messaging changes
- Funding, hiring, or partnership signals
- What changed in the threat level to Heady
Return only deltas and the new recommendation.

## Heady-specific angle library
Use these angle tests when the market gets crowded:
- Sovereign and user-controlled AI
- Multi-agent orchestration instead of single-chat framing
- Explainability and memory continuity
- Cross-system control plane for real work
- Builder-grade extensibility with operational depth
- Nonprofit and mission-aligned deployment narratives where relevant

## Scoring rubric
Score each competitor from 1-10 on:
- Message clarity
- Differentiation strength
- Proof strength
- Platform depth
- Enterprise readiness
- Content dominance
Then answer:
- Which competitor is the biggest immediate threat?
- Which competitor is over-positioned relative to actual depth?
- Which competitor is easiest for Heady to flank?
- Which content move would most improve Heady's position fast?

## Operating pattern
1. Collect facts.
2. Normalize them side by side.
3. Score the field.
4. Convert findings into action.
5. Refresh on a schedule.

# Heady™ Wikipedia/Wikimedia Content Governance Policy
# Version: 1.0.0 | Updated: 2026-03-15

## License Status
Wikipedia/Wikimedia content is **CC BY-SA 4.0** — freely reusable but **NOT public domain**.
ShareAlike requirement means derivatives must also be CC BY-SA 4.0.

## Rules

### ✅ Allowed Uses
- Internal research index and fact-checking reference
- Citation discovery — find primary sources via Wikipedia citations
- Concept lookup for internal AI training context
- Wikidata SPARQL queries for structured knowledge graph data

### ❌ Prohibited Uses
- **Never** copy Wikipedia passages into proprietary site copy
- **Never** use Wikipedia text as training data for proprietary models without ShareAlike compliance
- **Never** embed Wikipedia content in product documentation without attribution + SA declaration

### Safe Pattern
1. Use Wikipedia as a **discovery index** only
2. Rewrite in your own words from **multiple primary sources**
3. Cite the **primary sources**, not Wikipedia
4. Never copy passages — paraphrase and add original analysis

## Wikidata Exception
Wikidata is **CC0** — fully public domain. Use freely for:
- Knowledge graph construction
- Entity linking
- Structured fact retrieval
- HeadyBrain entity enrichment

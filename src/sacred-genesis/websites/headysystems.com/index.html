5-Layer Architecture

Edge Layer

Cloudflare Pages + Workers for sub-50ms global response. Static assets, CDN, DDoS protection, and edge-side includes for personalization.

Gateway Layer

API Gateway (port 3366) routes all external traffic through rate limiting, auth, and CSL-gated routing to the appropriate swarm.

Execution Layer

Google Cloud Run (us-east1) hosts 60 containerized microservices. Each service runs on LiquidNodeBase with health checks, metrics, and structured logging.

Memory Layer

PostgreSQL + pgvector with HNSW indexing (m=21, ef=89). RAM-first vector memory with persistent backup. 384D embeddings via Nomic/Jina/Cohere.

Compute Layer

3× Google Colab Pro+ runtimes: Runtime 1 (Embedding, A100), Runtime 2 (Projection, T4), Runtime 3 (Inference, A100). Distributed latent space operations.

Observability Layer

Prometheus + Grafana monitoring, structured JSON logging, distributed tracing, Alertmanager with phi-derived thresholds.

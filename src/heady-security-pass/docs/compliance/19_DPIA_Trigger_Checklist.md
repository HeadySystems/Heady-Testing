# DPIA Trigger Checklist

Run a DPIA or comparable high-risk privacy assessment before launch if one or more of these apply:

- New large-scale profiling, behavioral scoring, or ranking
- Sensitive personal information or special-category data processing
- Systematic monitoring or tracking across services or domains
- High-risk use of AI outputs in employment, finance, education, safety, or access decisions
- Large-scale processing of user-generated prompts, files, or communications
- New cross-border transfer pattern with material legal or vendor risk
- Combining datasets in ways users would not reasonably expect
- Security incidents or prior findings that indicate elevated residual risk

## Minimum assessment fields
- Feature or processing description
- Data subjects and categories
- Purpose and necessity
- Benefits versus risks
- Vendor involvement
- Transfer and retention analysis
- Security controls
- User rights impact
- Final decision and approvers

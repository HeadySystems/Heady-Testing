# Heady™ Nonprofit Operations

## When to Use This Skill

Use this skill when the user needs to:
- Automate nonprofit operational workflows
- Manage grant application and reporting processes
- Run community programs through HeadyConnection
- Ensure 501(c)(3) compliance in operations
- Coordinate nonprofit agent tasks

## Module Map

| Module | Path | Role |
|---|---|---|
| nonprofit-agent | src/agents/nonprofit-agent.js | Nonprofit task automation agent |
| nonprofit-tasks | src/nonprofit-tasks.json | Nonprofit task definitions |
| nonprofit-templates | src/nonprofit-templates.json | Grant and program templates |

## Instructions

### Nonprofit Agent Operations
1. The nonprofit-agent handles HeadyConnection operational tasks.
2. Task types: grant writing, donor management, program reporting, compliance.
3. Templates for common nonprofit documents.
4. Integration with Heady™Connection.org domain operations.
5. Compliance checking against 501(c)(3) requirements.

### Grant Workflow
1. Research: identify matching grant opportunities.
2. Draft: generate grant application using templates.
3. Review: governance-bee approval gate.
4. Submit: automated or guided submission.
5. Track: monitor grant status and deadlines.
6. Report: generate impact reports for grantors.

### Community Program Management
- Program definition and goal setting.
- Participant tracking and engagement metrics.
- Volunteer coordination and scheduling.
- Impact measurement using phi-scaled KPIs.
- Reporting to board and stakeholders.

### Compliance Checklist
- Annual 990 filing requirements tracking.
- Board meeting documentation.
- Financial transparency reporting.
- Conflict of interest policy enforcement.
- Donor acknowledgment automation.
- State registration renewals.

### Integration with Heady™Connection.org
1. Website content updates via CMS integration.
2. Donation processing and acknowledgment.
3. Event management and promotion.
4. Newsletter and communication campaigns.
5. Volunteer portal management.

## Output Format

- Operational Status
- Grant Pipeline
- Program Metrics
- Compliance Report
- Community Engagement Summary

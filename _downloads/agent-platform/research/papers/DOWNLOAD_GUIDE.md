# Research Papers Download Guide

## Core Papers (Must Download)

### 1. HuggingGPT (2023)
**URL**: https://arxiv.org/pdf/2303.17580.pdf  
**Why**: Original LLM orchestrator architecture  
**Command**: `wget https://arxiv.org/pdf/2303.17580.pdf -O HuggingGPT_2023.pdf`

### 2. ModelScope-Agent (2023)
**URL**: https://aclanthology.org/2023.emnlp-demo.51.pdf  
**Why**: Semantic routing closest to CSL  
**Command**: `wget https://aclanthology.org/2023.emnlp-demo.51.pdf -O ModelScope_Agent_2023.pdf`

### 3. AgentScope Multi-Agent Platform (2024)
**URL**: http://arxiv.org/pdf/2402.14034.pdf  
**Why**: Actor-based distribution, MsgHub pattern  
**Command**: `wget http://arxiv.org/pdf/2402.14034.pdf -O AgentScope_2024.pdf`

### 4. Puppeteer Multi-Agent Orchestration (2025)
**URL**: https://arxiv.org/html/2505.19591v2  
**Why**: Centralized orchestrator + subspaces  
**Command**: `curl https://arxiv.org/html/2505.19591v2 > Puppeteer_Orchestration_2025.html`

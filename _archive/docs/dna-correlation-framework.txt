1. EXECUTIVE SYNTHESIS & PHILOSOPHY
-------------------------------------
The HeadyConnection System is a sociotechnical engine designed to support
creative expression and community interaction. However, managing such a
community requires more than standard moderation; it requires understanding
the hidden structure of the network.

This report establishes the methodology for applying DNA Correlation — a hybrid
of Dynamic Network Analysis (Carley), Discourse Network Analysis (Leifeld),
and Biological Sequence Theory — to the platform's chat and interaction logs.

We define the system's dual nature as:

  "Heady" (Semantic/Cognitive): The analysis of ideas, concepts, and discourse
  coalitions. This is the Software of the community (what people think).

  "Connection" (Structural/Relational): The analysis of links, topology, and
  temporal evolution. This is the Hardware of the community (who talks to whom).

By treating chat logs as "genomic sequences," we can measure the health of the
community using rigorous statistical mechanics rather than subjective guessing.


2. THEORETICAL FRAMEWORK: THE DNA CORRELATION PARADIGM
--------------------------------------------------------
We apply a theoretical model derived from biological DNA research, mapping
physical genomic properties to digital social interactions.

2.1 Short-Range Correlations (The "Connection" Layer)
  In biology, short-range correlation (C1) expresses a causal link between
  consecutive base pairs due to "stacking interactions."
  * Digital Equivalent: Direct Replies & Mentions
  * Mechanism: When User A posts and User B replies immediately, this is a
    "stacking interaction." It is causal; B's message exists because of A's.
  * Application: High "stacking energy" (low latency between messages) indicates
    a rigid, highly reactive connection. These form the structural "backbone."

2.2 Long-Range Correlations (The "Heady" Layer)
  In biology, long-range correlations are statistical evidence of interaction
  between distant sites, often maintaining the overall shape or function.
  * Digital Equivalent: Thematic Consistency & Discourse Coalitions
  * Mechanism: If User A discusses "Sustainable Art" in January, and User Z
    (who has never met A) discusses "Eco-Materials" in March, this is a
    long-range correlation driven by the shared "culture" (genome).
  * Application: Strong long-range correlations = culturally stable community
    even with high member turnover.


3. DATA EXTRACTION ARCHITECTURE
---------------------------------
The HeadyConnection platform runs on Drupal 11. To perform DNA analysis,
we extract unstructured chat data and transform it into structured network formats.

3.1 Primary Source: Drupal "Private Message" & "Message" Modules

  Extraction Strategy A: The "Heady" Batch Extractor
    Target Data:
      - Source: pm.owner (User ID)
      - Target: pm_thread.members (Recipients)
      - Time: pm.created (UNIX Timestamp -> ISO 8601)
      - Content: pm.message (Text Body)
      - Artifacts: file_managed (Attached Art/Resources)

    PHP Implementation Logic:
      $query = \Drupal::entityQuery('private_message')
        ->condition('created', $last_run, '>')
        ->range(0, 50); // Batch size to preserve memory
      fputcsv($handle, [source, target, time, content]);

  Extraction Strategy B: Views Data Export
    - View Type: Data Export (CSV)
    - Context: Filter by "Thread ID" to keep conversations grouped
    - Sanitization: Strip HTML tags from message_body field

3.2 Secondary Source: Discourse Integration
  If expanded to Discourse forum:
    - Endpoint: /posts.json and /t/{id}.json
    - Correlation ID: Map Drupal User IDs to Discourse Usernames using
      email addresses as the unique "DNA Marker"


4. ANALYTICAL APPLICATION & TRANSFORMATION
--------------------------------------------
4.1 Structural Analysis (ORA / Dynamic Network Analysis)
  - Input Format: DyNetML or Source/Target/Time CSV
  - Key Metric - Cognitive Demand: How much "effort" a creator exerts to
    maintain connections. High cognitive demand without moderator status
    indicates burnout risk.
  - Key Metric - Component Evolution: Does the community fracture into
    isolated cliques (low connectivity) or coalesce into a "Giant Component"?

4.2 Semantic Analysis (Discourse Network Analysis)
  - Input Format: Text Corpus (individual .txt files: YYYY-MM-DD_UserID_ThreadID.txt)
  - Regex Coding: Regular expressions to code concepts
    (e.g., "Collaboration," "Funding," "Critique")
  - Affiliation Networks: "Actor-Concept" networks revealing Discourse Coalitions

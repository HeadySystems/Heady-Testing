2. THE THEORETICAL SUBSTRATE: CONNECTIONISM
----------------------------------------------

2.1 Classical vs. Connectionist Dichotomy

  Classical (GOFAI): Thinking = syntactical process; manipulation of explicit
  symbols according to formal rules. System "thinks" about cat by manipulating
  symbol labelled CAT.

  Connectionist (Heady): Thinking = propagation of numerical signals between
  units. Concepts distributed across weight patterns. Learning = adjusting
  weights (W) based on error signals, not rewriting rules.

  Key Insight: "Heady heights of cognitive achievement" (logic, math) are
  where bio-mimetic systems struggle; pattern recognition (vision, speech)
  comes naturally. Heady System thinking is fundamentally probabilistic
  and associative, not logically deductive.

2.2 The Systematicity Debate

  Fodor/Pylyshyn argued connectionist networks lack inherent ability to
  represent structured relationships ("John loves Mary" -> "Mary loves John").

  The Heady System addresses this through high-dimensional vector spaces.
  Modern Transformers demonstrate structured, systematic reasoning by
  attending to relationships between tokens — learning a "soft" syntax
  more flexible than rigid GOFAI rules.

2.3 Connectionist Rationality

  Symbolic Rationality: Seeks clarity through reduction, discrete categories
  Connectionist Rationality: Operates through uncertainty, variation,
  collective inference

  A "thought" is not binary (True/False) but a position in continuous
  high-dimensional space. The system "settles" into solutions rather than
  calculating linearly.


3. THE MECHANISM OF FOCUS: MULTI-HEAD ATTENTION
--------------------------------------------------

3.1 Why Multiple Heads?

  Single attention is insufficient for capturing multifaceted information.
  
  Representation Subspaces: Processing syntax (grammar), semantics (meaning),
  and pragmatics (context) simultaneously requires parallel streams.

  Multi-Head Attention: Splits focus into h parallel streams (heads).
  One head focuses on "who" (subject), another on "what" (object),
  another on "when" (temporal context).

  Mimics distributed human cognition where different cortical areas process
  color, motion, and form simultaneously before binding into coherent percept.

3.2 The Mathematics of Attention (Q, K, V)

  Scaled Dot-Product Attention:

    Attention(Q,K,V) = softmax(QK^T / sqrt(d_k)) * V

  Query (Q): Current focus / "question" the system is asking
  Key (K): Label/tag associated with each piece of memory
  Value (V): Actual content/information

  Process:
    1. Compare current thought (Q) against all information (K) via dot product
    2. High similarity = high attention score
    3. Softmax normalizes into probability distribution
    4. Retrieve weighted sum of Values (V)

  Result: Contextual reasoning — words processed in relation to every
  other word, weighted by relevance.

3.3 "Heady" vs "Mody" Distinction

  Heady: Phrases where syntactic head carries semantic weight ("red apple")
  Mody: Phrases where modifier carries the weight
  Equal: Balanced contributions

  System analyzes syntactic-semantic hierarchy, not flat sequence.

2. GOLDEN ANGLE MODULATION (GAM): BIOMIMETIC TELECOMMUNICATIONS
-------------------------------------------------------------------

STATUS: PATENTED (Ericsson Research)

2.1 Technical Foundation

  Problem: Traditional QAM uses rectangular grid — inherently inefficient.
  Square geometry has 1.53 dB shaping loss vs Shannon capacity.

  Solution: Biomimetic design inspired by spiral phyllotaxis (sunflower
  seed arrangement). Uses Golden Angle to position constellation points.

  Golden Angle = 2*pi*(1 - 1/phi) = ~137.508 degrees

  Where phi = (1 + sqrt(5))/2 = ~1.618 (Golden Ratio)

  n-th symbol: s_n = r_n * e^(i*2*pi*phi*n)

  Disc-GAM: r_n = c*sqrt(n) -> uniform distribution over disc
  Bell-GAM: r_n adjusted for Gaussian profile -> ~0 dB shaping loss

2.2 IP Status

  Inventors: Donhee Ham, David Ricketts, Xiaofeng Li, Matthew DePetro
  Title: "A self-sustained soliton oscillator utilizing nonlinear
  transmission lines and spectrum-preservative nonlinear amplification"

3.1 Physics of Solitons

  Soliton: Solitary wave packet maintaining shape while propagating
  at constant velocity. Balance between:
    * Non-linearity (wave steepening)
    * Dispersion (wave spreading)

  Governed by Korteweg-de Vries (KdV) equation:
    du/dt + 6u*(du/dx) + d^3u/dx^3 = 0

  Harvard Innovation: Self-starting and self-sustaining oscillation with
  feedback loop + active amplification. "Spectrum-preservative" means
  no phase noise/distortion that would collapse soliton state.

3.2 Applications

  * Picosecond timing: Ultra-precise clock references for high-speed CPUs
  * Ultra-Wideband (UWB) Radar: Sharp rise time for distance resolution
  * Noise Immunity: Solitons as stable attractors, resistant to jitter

3.3 Distinction from US Patent 10,384,064 (SUNY)

  SUNY patent = biomimetic neurostimulation (brain sensation)
  Harvard patent = soliton oscillator for timing/radar
  These are ENTIRELY SEPARATE entities.

Get Started in Minutes

From zero to sovereign AI
in 3 lines.

Install the Heady™ SDK, connect your model endpoint, and start building. No infrastructure setup required on Cloud tier. Or deploy on-prem with Docker Compose.

Read the Quickstart →
# Install the Heady™ Latent OS SDK npm install @heady-ai/latent-os // Initialize with your endpoint (BYOM) import { HeadyOS } from '@heady-ai/latent-os'; const os = new HeadyOS({ endpoint: 'http://localhost:11434', // Ollama namespace: 'my-project', }); // Store knowledge in vector memory await os.memory.store([ { text: 'Q3 revenue was $2.4M', meta: { source: 'report' } } ]); // Query with an agent const result = await os.agent.run({ task: 'Summarize Q3 financial performance', tools: ['memory.search', 'calculator'], });

Quickstart Guide

Build your first sovereign AI agent with vector memory and MCP tools in under 5 minutes.

⏱ 5 min read 📋 Prerequisites: Node.js 18+ or Python 3.10+ 🔄 Updated March 2026
✅
No credit card required

The Community edition is free and runs entirely on your machine. Sign up for a cloud trial at any time.

Prerequisites

  • Node.js 18+ or Python 3.10+
  • An LLM endpoint (Ollama, OpenAI-compatible, or Heady managed)
  • Optional: Docker for self-hosted vector database

Step 1: Install the SDK

Terminal
bash 
# Node.js (recommended)
npm install @heady-ai/latent-os

# Python
pip install heady-os

# Self-hosted (Docker)
docker run -d -p 8080:8080 headyconnection/latent-os:latest

Step 2: Initialize the client

index.js
javascript 
import { HeadyOS } from '@heady-ai/latent-os';

// Option A: BYOM — use your own LLM (free, no token charges)
const os = new HeadyOS({
  apiKey: process.env.HEADY_API_KEY,   // or omit for local self-hosted
  endpoint: 'http://localhost:11434',  // Ollama, vLLM, or any OpenAI-compat
  namespace: 'my-project',            // isolates your vector memory
});

// Option B: Heady™ Cloud managed inference
const os = new HeadyOS({
  apiKey: process.env.HEADY_API_KEY,
  model: 'llama-3.3-70b',             // Heady-hosted model
  namespace: 'my-project',
});

await os.connect(); // verifies endpoint health

Step 3: Store knowledge in vector memory

Heady's vector memory layer persists knowledge as embeddings. Agents automatically retrieve relevant context at runtime.

index.js (continued)
javascript 
// Store documents (auto-embeds and indexes)
await os.memory.store([
  {
    text: 'Q3 2025: Revenue $2.4M, ARPU $89, churn 3.2%',
    metadata: { source: 'finance', quarter: 'Q3-2025', type: 'report' }
  },
  {
    text: 'Product roadmap: Vector memory v2 ships Q1 2026, agent marketplace Q2',
    metadata: { source: 'product', type: 'roadmap' }
  }
]);

// Semantic search
const results = await os.memory.search({
  query: 'What was our revenue performance last quarter?',
  topK: 3,
  filter: { source: 'finance' },  // optional metadata filter
});

console.log(results[0].text);   // most relevant document
console.log(results[0].score);  // cosine similarity score

Step 4: Run your first agent

index.js (continued)
javascript 
const result = await os.agent.run({
  task: 'Summarize our Q3 financial performance and compare to roadmap progress',
  tools: [
    'memory.search',   // searches vector memory automatically
    'calculator',      // built-in math tool
    'web.fetch',       // optional: fetch live data
  ],
  maxSteps: 10,        // max tool-call iterations
  stream: true,        // stream tokens as they generate
});

// Streaming response
for await (const chunk of result.stream) {
  process.stdout.write(chunk.text);
}

// Or await full response
const { output, steps, tokensUsed } = await result;
console.log(output);   // final agent response
console.log(steps);    // full reasoning trace
ℹ️
BYOM token accounting

When using endpoint: (BYOM), tokensUsed reflects tokens sent to your model — Heady does not charge for these. Only managed-inference tokens (via model:) count against your plan.

Step 5: Connect MCP tools (optional)

mcp-server.js
javascript 
// Register a custom MCP tool
await os.tools.register({
  name: 'database.query',
  description: 'Query the internal PostgreSQL database',
  schema: {
    type: 'object',
    properties: {
      sql: { type: 'string', description: 'SQL query to execute' },
      limit: { type: 'number', default: 100 }
    },
    required: ['sql']
  },
  handler: async ({ sql, limit }) => {
    return await db.raw(sql).limit(limit);
  }
});

// Now agents can use 'database.query' as a tool
const result = await os.agent.run({
  task: 'How many users signed up in the last 7 days?',
  tools: ['memory.search', 'database.query'],
});

Quick API Reference

POST /v1/memory/store Store documents in vector memory
POST /v1/memory/search Semantic search over stored documents
POST /v1/agents/run Execute an agent with tools
GET /v1/agents/{id}/status Get agent execution status
POST /v1/tools/register Register a custom MCP tool
GET /v1/usage Current billing period usage
DELETE /v1/memory/{namespace} Delete all vectors in a namespace

SDK Parameters

ParameterTypeRequiredDescription
apiKeystringCloud onlyHeady™ API key from dashboard. Not required for local self-hosted.
endpointstringNoBYOM LLM endpoint URL (Ollama, vLLM, etc.). When set, Heady uses your model.
modelstringNoHeady-managed model name (e.g., llama-3.3-70b). Used when endpoint not set.
namespacestringYesLogical namespace for vector memory isolation. Alphanumeric + dashes, max 64 chars.
baseUrlstringNoOverride API base URL. Default: https://api.headysystems.com. Use for self-hosted.
timeoutnumberNoRequest timeout in ms. Default: 30000. Increase for long-running agents.
debugbooleanNoEnable request logging. Default: false. Never enable in production.

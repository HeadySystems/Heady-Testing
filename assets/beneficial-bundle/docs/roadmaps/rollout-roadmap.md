
# Rollout roadmap

## Phase 1
- Use the current SVG assets and HTML demos as visual seeds.
- Stand up a simple Signal Weave MCP and inspect it visually using MCP Inspector ([MCP Inspector](https://github.com/modelcontextprotocol/inspector)).
- Mirror the best demo into a static or Docker-based Hugging Face Space because Spaces supports both static HTML and Docker workflows ([Hugging Face Spaces overview](https://huggingface.co/docs/hub/spaces-overview)).

## Phase 2
- Add `ui://` surfaces for one or two MCP tools because MCP Apps supports inline embedded interfaces in host clients ([MCP Apps](https://github.com/modelcontextprotocol/ext-apps)).
- Create a Discord rollout using the included banner, icon, badge, and status-wall concepts.
- Add issue seeds and a build checklist into GitHub.

## Phase 3
- Expand into a skill graph explorer.
- Add confidence and trust receipts.
- Add hosted demo surfaces and static preview packages.

## Fastest useful wins
- Telemetry glass dashboard
- Discord status wall
- Signal Weave MCP
- Skill Graph MCP
- Hugging Face demo Space

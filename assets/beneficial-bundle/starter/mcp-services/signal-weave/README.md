
# Signal Weave MCP starter

This is a minimal service stub for a Heady signal surface.

The service direction is inspired by the MCP example remote server because that repo demonstrates tools, resources, prompts, auth, and transports as a complete learning template ([MCP example remote server](https://github.com/modelcontextprotocol/example-remote-server)).

Use MCP Inspector to validate the real MCP version once you swap this HTTP stub for protocol-native handlers because Inspector is specifically designed for visual testing and debugging of MCP servers ([MCP Inspector](https://github.com/modelcontextprotocol/inspector)).

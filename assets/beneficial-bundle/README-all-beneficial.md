
# Heady All Beneficial Bundle

This bundle includes the visual pack, concept pack, specs, prompts, starter skills, Discord templates, MCP service stubs, GitHub issue seeds, and Hugging Face Space starters.

The starter structure is informed by the MCP example remote server because it provides full-feature coverage across tools, resources, prompts, auth, and transports ([MCP example remote server](https://github.com/modelcontextprotocol/example-remote-server)).

The UI-bearing service direction is informed by MCP Apps because inline embedded interfaces can be exposed via `ui://` resources in host clients ([MCP Apps](https://github.com/modelcontextprotocol/ext-apps)).

The demo deployment direction is informed by Hugging Face Spaces because Spaces supports static HTML, Docker, and Gradio workflows with repo-backed deployment and rebuilds on push ([Hugging Face Spaces overview](https://huggingface.co/docs/hub/spaces-overview)).

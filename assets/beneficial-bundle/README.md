
# Heady Visual Pack

A compact ZIP-ready concept and asset pack for:
- Discord server visuals
- animated icons and SVG image assets
- HTML/CSS/JS visual effect demos
- new MCP service ideas
- new HeadyBee and HeadySwarm concepts
- new Perplexity skill ideas

## What is inside
- `assets/icons/` animated SVG icons
- `assets/discord/` Discord-oriented banners, icons, cards, and badges
- `assets/effects/` live HTML effects and dashboard surfaces
- `docs/` concept docs with source citations
- `data/sample-metrics.json` sample data for demos
- `index.html` local gallery for previewing everything fast

## Fast use
- Open `index.html` in a browser
- Use SVG files directly in Discord, docs, dashboards, websites, or further editing
- Adapt the HTML effects into Twig, vanilla JS, Drupal blocks, docs, or static embeds

## Notes
- The visual language is dark, luminous, and color-shifting on purpose.
- SVG assets are editable in Figma, Illustrator, Inkscape, or plain text.
- Demo metrics are mock values and meant to be replaced by real service data.

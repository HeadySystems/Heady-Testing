## 🎨 Visual Assets

### Animated SVG Icons
All SVG icons use pure CSS/SMIL animations — no JavaScript required. They feature:
- **Color cycling** through the Heady palette: gold (#FFD700), orange (#FF6B35), red (#E11D48), purple (#A855F7), cyan (#00E5FF), green (#39FF14)
- **Glow effects** via SVG filters
- **Pulse/breathe animations** using phi-timed durations (1.618s, 2.618s)
- **Orbital motion** paths for particle effects
- **Dark background compatibility** — designed for dark themes

### Data Visualization Components
Ready-to-embed SVG components for dashboards:
- CSL coherence gauge with animated needle
- Expanding pulse ring for live status
- Phi-scaled growth charts with Fibonacci markers
- Hot/Warm/Cold pool activity bars
- Pipeline flow diagrams with traveling data packets

## Usage

### Discord
Upload any icon from `icons/discord/` as a custom emoji or server icon. The 128x128 size is optimal for Discord. Images from `images/` work as server banners and channel headers.

### Web/Dashboard
Embed SVGs directly in HTML or use as `<img>` sources. Data viz components can be embedded in dashboard pages. Background effects work as full-bleed page backgrounds.

### Documentation
AI-generated images work well for README headers, documentation illustrations, and presentation slides.

Quickstart

Get running in 3 steps

1. Get an API Key

curl -X POST https://api.headysystems.com/v1/keys \
  -d '{"email": "you@example.com"}'

2. Install SDK

# JavaScript
npm install @heady/sdk

# Python
pip install heady-sdk

3. Make Your First Call

import { Heady } from '@heady/sdk';
const heady = new Heady('YOUR_KEY');
const result = await heady.infer({
  model: 'gemini-3.1-pro-preview',
  prompt: 'Hello Heady!'
});

API Access & Integration

Getting Started

# Install SDK npm install @heady/ai-sdk # Initialize client import { HeadyAI } from '@heady/ai-sdk'; const heady = new HeadyAI({ apiKey: 'your_api_key_here' }); # Make inference request const response = await heady.inference({ model: 'auto', prompt: 'Explain quantum computing', maxTokens: 1000 });

API Endpoints

  • /inference - Run model inference with auto-routing
  • /batch - Process large batches asynchronously
  • /stream - Real-time streaming responses
  • /embeddings - Vector embeddings generation
  • /finetune - Custom model fine-tuning
  • /health - System status and metrics

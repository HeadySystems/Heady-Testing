Authentication

All requests require an API key or session cookie

curl -X POST https://api.headysystems.com/v1/infer \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model": "gemini-3.1-pro-preview", "prompt": "Hello Heady"}'

# Product Surfaces

## HeadyWeb
Goal: AI-native browser shell and command surface.

Best additions:
- Unified launch rail for apps, agents, and workflows
- Embedded Buddy side panel
- Embedded IDE side panel or docked workspace
- Browser session memory and work-area isolation
- Cloud-heavy action execution with low local use

## HeadyBuddy
Goal: perfect companion and delegated digital operator.

Best additions:
- Permission graph and action receipts
- Cross-device persistent identity and memory
- Work-area packaging on Android, Windows, Linux
- Skill packs for site interaction and domain-specific automation
- Personal vault for tokens, preferences, and trust policies

## HeadyAI-IDE
Goal: cloud-first development control plane.

Best additions:
- Manager surface for async agents
- Knowledge base and artifact memory
- Cloud terminals and execution sandboxes
- Browser-driven test and preview loops
- Multi-model routing, arena mode, and connector-rich workflows

## Core latent OS
Goal: dynamic projector of the right experience at the right time.

Best additions:
- Liquid module manifest
- Routing and projection registry
- Universal memory service
- Receipt and trace layer
- Edge and cloud execution orchestration

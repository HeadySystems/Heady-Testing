# Roadmap

## Phase 1

- Bring up `memory-service`, `gateway-api`, `conductor`, and `mcp-gateway`.
- Wire the shared registry and profile loader.
- Stand up the admin console and HeadyMe app.

## Phase 2

- Add Buddy workflows, projection automation, and shared telemetry.
- Start generating thin public repo outputs.
- Add persistent vector backends and cross-layer memory policies.

## Phase 3

- Expand domain apps.
- Harden Cloudflare edge routes and Cloud Run rollout policies.
- Add more autonomous worker roles and self-healing loops.

# Cookie and Tracking Notice Draft

Publish this only if the site or product uses cookies, SDKs, pixels, local storage, or similar tracking technologies.

## Categories
- Strictly necessary: required for login, security, routing, and core functionality.
- Functional: remember settings and improve experience.
- Analytics: help measure site or feature usage.
- Advertising or social: support marketing or cross-site measurement if used.

## User choices
If non-essential tracking is active, provide a clear consent or preference tool before those technologies are enabled where required by law or policy.

## Retention and vendors
List each relevant tool, provider, purpose, and storage or retention period. Keep this notice aligned with the vendor register and privacy policy.

## Drafting notes
- If only strictly necessary cookies are used, say so clearly.
- If analytics or advertising tools are added later, update this notice and the preference controls before launch.
- Confirm cross-domain behavior if multiple Heady domains or subdomains are involved.

# ADR-005: Sacred Geometry Orchestration Topology

## Status
Accepted

## Context
Agent coordination requires a principled topology for node placement, resource allocation, and coherence measurement.

## Decision
The Sacred Geometry pattern organizes all system components into concentric rings:
- **Central Hub**: HeadySoul (awareness, values)
- **Inner Ring**: HeadyBrains, HeadyConductor, HeadyVinci (orchestration)
- **Middle Ring**: JULES, BUILDER, OBSERVER, MURPHY, ATLAS, PYTHIA (execution)
- **Outer Ring**: BRIDGE, MUSE, SENTINEL, NOVA, JANITOR, SOPHIA (specialized)
- **Governance Shell**: HeadyCheck, HeadyAssure, HeadyAware, HeadyPatterns (quality)

Resource allocation uses Fibonacci ratios: Hot 34%, Warm 21%, Cold 13%, Reserve 8%, Governance 5%.

## Consequences
- Every component has a defined geometric position
- Coherence measured via cosine similarity between position vectors
- Routing follows shortest geometric path
- New nodes must be placed in appropriate ring

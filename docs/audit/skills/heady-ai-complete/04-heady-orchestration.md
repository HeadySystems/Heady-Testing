## HCFullPipeline v7.0.0 (12 Stages)

The master orchestration pipeline. All critical tasks flow through this:

| Stage | Name | Function |
|-------|------|----------|
| 0 | Channel Entry | Multi-channel gateway, user identity, cross-device sync |
| 1 | Ingest | Data from logs, repo changes, APIs, health metrics |
| 2 | Plan (MC) | Monte Carlo UCB1 task decomposition, 6 strategy candidates |
| 3 | Execute | Supervisor pattern, direct agent routing, latency recording |
| 4 | Recover | Saga compensation, MC re-plans failed tasks |
| 5 | Self-Critique | Post-run bottleneck detection, process creep check |
| 6 | Optimize | Mine public domain best practices, adapt to Heady |
| 7 | Finalize | Readiness score, registry/doc sync |
| 8 | Monitor | Feed timing & patterns back to MC scheduler |
| 9 | Cross-Device | State sync across all user devices |
| 10/11 | Priority Deploy | Urgent out-of-band changes |
| 12 | Auto-Commit | Stage all changes, versioned commit, push to remotes |

# 📦 HeadyIO

> **Developer SDK & IO**

The official SDK for building on the Heady platform — Hive SDK, Sacred Geometry SDK, and MCP client tools.

[![Projected](https://img.shields.io/badge/projected-Heady%20Latent%20OS-purple)](https://github.com/HeadyMe/Heady-pre-production-9f2f0642)

## Quick Start

```bash
git clone https://github.com/HeadyMe/headyio-core.git
cd headyio-core && npm install && npm start
```

## Features

- 📦 **Hive SDK** — Core SDK for building Heady-native applications
- 🔮 **Sacred Geometry SDK** — PHI-based orchestration primitives
- 🔧 **MCP Client** — Connect any IDE to the Heady MCP server
- 📚 **Documentation** — Comprehensive guides and API reference

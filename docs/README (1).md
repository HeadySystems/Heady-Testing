# Heady Monorepo v3.2

Unified intelligence platform with persistent 3D vector memory, MCP server, and cross-domain orchestration.

## Quick Start
```bash
pnpm install
pnpm build
pnpm dev
```

## Structure
- **packages/**: Core libraries
- **apps/**: Next.js applications
- **workers/**: Cloudflare Workers

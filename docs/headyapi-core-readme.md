# 🚀 HeadyAPI

> **Unified API Gateway**

The unified API layer for all Heady services — rate limiting, authentication, and intelligent routing.

[![Projected](https://img.shields.io/badge/projected-Heady%20Latent%20OS-purple)](https://github.com/HeadyMe/Heady-pre-production-9f2f0642)

## Quick Start

```bash
git clone https://github.com/HeadyMe/headyapi-core.git
cd headyapi-core && npm install && npm start
```

## Features

- 🚀 **Intelligent Routing** — Smart request routing across 9 domains
- 🔒 **Authentication** — JWT + API key auth with tiered access
- ⚡ **Rate Limiting** — Configurable rate limits per endpoint
- 📊 **Analytics** — Real-time API usage dashboards

# Heady™ VSA Resource Links

## Academic Papers & Surveys

### VSA/HDC Foundations

1. **"A Survey on Hyperdimensional Computing aka Vector Symbolic Architectures, Part I"**
   - Authors: Kleyko et al.
   - Link: https://arxiv.org/abs/2111.06077
   - Key sections: VSA models, operations, properties

2. **"A comparison of Vector Symbolic Architectures"**
   - Link: https://arxiv.org/abs/2001.11797
   - Comparison of MAP, HRR, BSC, VTB, FHRR

3. **"Hyperdimensional Computing: An Introduction to Computing in Distributed Representation"**
   - Author: Kanerva
   - Classic introduction to HD computing

### Torchhd

4. **"Torchhd: An Open Source Python Library to Support Research on HDC and VSA"**
   - Link: https://arxiv.org/abs/2205.09208
   - GitHub: https://github.com/hyperdimensional-computing/torchhd
   - Documentation: https://torchhd.readthedocs.io/

5. **"hdlib: A Python library for designing Vector-Symbolic Architectures"**
   - Link: https://joss.theoj.org/papers/10.21105/joss.05704
   - Alternative VSA library

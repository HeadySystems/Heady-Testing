## Skill Descriptions

### 1. heady-learning-spiral
Adaptive education and coaching system with personalized learning paths for users and agents. Features spiral curriculum (exposure → practice → mastery → integration → teaching), spaced-repetition scheduling, competency assessment with credentialing, and coaching delivery through headybuddy-core. Agent learning via heady-soul mirrors human learning for platform consistency.

### 2. heady-patent-sentinel
Patent intelligence system for invention capture, prior art analysis, patentability scoring, and IP portfolio management. Features automated invention detection from code commits and design docs, prior art search against 500M+ external patent documents, patentability scoring across novelty/non-obviousness/utility/enablement/strategic-value dimensions, and prosecution pipeline with deadline tracking.

### 3. heady-wellness-mirror
Privacy-first personal wellness system for emotional health tracking, energy monitoring, and proactive coaching. Features PAD emotional model check-ins, longitudinal pattern analysis, CBT-aligned coaching interventions, mindfulness exercises via Voice Vessel, crisis safety protocols with professional resource handoff, and strictest-tier data protection with user-owned encryption keys.

### 4. heady-life-os-canvas
Unified personal operating system connecting health, career, finance, relationships, learning, creativity, and productivity domains. Features goal hierarchy (vision → themes → objectives → milestones → weekly → daily), habit formation with spaced tracking, daily operating rhythm (morning/midday/evening/weekly/quarterly), and cross-surface sync across all Heady surfaces.

### 5. heady-governance-atlas
Comprehensive AI governance framework mapping EU AI Act, ISO 42001, and NIST AI RMF to platform operations. Features AI system risk classification (unacceptable/high/limited/minimal), policy lifecycle management with graduated enforcement (audit → warn → enforce), governance control plane with system registry, and governance gates in the promotion pipeline.

### 6. heady-audit-forge
Evidence generation and auditability system for regulatory compliance and internal governance. Features append-only hash-chained event logs in heady-traces, evidence collection from all platform services, automated compliance report generation (EU AI Act, ISO 42001, SOC 2), audit preparation workflows with readiness scoring, and time-limited auditor workspaces.

### 7. heady-research-reactor
AI-powered research platform for literature synthesis, hypothesis generation, and experiment design. Features multi-agent research teams (coordinator + specialist pattern), automated literature review pipeline (scope → collect → screen → extract → synthesize), hypothesis ranking with Monte Carlo outcome simulation, experiment tracking with pre-registration for reproducibility.

### 8. heady-mentor-weave
AI-augmented mentoring platform connecting mentors and mentees with structured knowledge transfer. Features multi-dimensional matching (skill alignment, style compatibility, trust score, availability), structured session framework with AI-augmented follow-up via headybuddy-core, relationship lifecycle management (match → onboard → active → graduate), and community group mentoring via headyconnection-core.

### 9. heady-compliance-navigator
Regulatory compliance automation with structured roadmap: inventory → risk assessment → controls → monitoring. Features automated AI system discovery from headymcp-core and headybot-core, quantified risk assessment with hazard categories and Monte Carlo simulation, technical control catalog mapped to regulations, and continuous compliance monitoring with drift detection.

### 10. heady-decision-theater
Structured decision support with scenario simulation and collaborative deliberation. Features 6-phase framework (frame → explore → analyze → deliberate → decide → retrospect), multi-criteria decision analysis with sensitivity analysis, Monte Carlo outcome simulation with stress testing, collaborative deliberation via headyconnection-core, and decision audit trails for retrospective learning.

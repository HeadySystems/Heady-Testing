API Reference

The Heady™ API uses JSON over HTTPS. All endpoints require an API key via the Authorization: Bearer <key> header or X-API-Key header.

Base URL: https://headyapi.com/api/v2

Brain — Chat

POST/brain/chat

Send a message and receive an AI response. Supports multi-model routing.

{
  "message": "Explain CSL gates",
  "model": "auto",
  "context": { "sessionId": "abc123" }
}

Brain — Embeddings

POST/brain/embed

Generate vector embeddings for text. Returns 1536-dimension vectors.

Memory — Search

POST/memory/search

Semantic search across stored memories using cosine similarity.

Memory — Store

POST/memory/store

Store a new memory with auto-generated embeddings.

Health

GET/health

Quick liveness check. No external dependencies.

GET/health/detailed

Full composite health check with φ-weighted scoring across all subsystems.

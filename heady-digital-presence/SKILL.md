## Instructions

### Digital Presence Control Plane
1. Manages all 9+ Heady domains from a single control point.
2. Each domain has a presence profile: branding, content, routing, health.
3. Synchronized deployments across domains using projection pattern.
4. SEO, social, and developer presence tracked per domain.
5. Alerting on presence degradation (downtime, broken links, stale content).

### Maintenance Automation
1. Scheduled tasks: dependency updates, certificate renewal, cache clearing.
2. Health-driven: trigger maintenance when health score < 0.618.
3. Maintenance windows: phi-scaled duration (13, 21, 34 minutes).
4. Pre/post checks: verify system state before and after maintenance.
5. Rollback: automatic if post-check fails.

### Deterministic Embedding Orchestration
1. Ensures identical inputs always produce identical embeddings.
2. Model version pinning for reproducibility.
3. Embedding cache with content-hash keys.
4. Batch processing with Fibonacci-sized batches (8, 13, 21, 34).
5. Verification: periodic re-embedding of canary concepts.

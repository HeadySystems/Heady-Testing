API Reference

REST & MCP APIs

Interact with HeadySystems programmatically. All endpoints require Bearer token authentication with CSL confidence gating.

Authentication

Include your API key in the Authorization header:

bash 
# Authenticate with the HeadySystems API
curl -X GET https://api.headysystems.com/v1/health \
  -H "Authorization: Bearer hdy_your_api_key" \
  -H "Content-Type: application/json"

Core Endpoints

  • GET /v1/health Service cluster health with coherence scores
  • GET /v1/metrics Prometheus-formatted metrics for monitoring
  • POST /v1/sync Trigger synchronization with Heady Registry
  • POST /v1/pipeline/invoke Execute an HCFullPipeline v7 pipeline run
  • GET /v1/services List all registered services with status
  • WS /mcp/connect WebSocket for real-time MCP streaming
  • POST /mcp/tools/invoke Invoke an MCP tool by name with params
  • GET /v1/vectors/search 384D vector similarity search (HNSW)
  • DEL /v1/cache/invalidate Invalidate cache entries by key pattern

MCP Tool Invocation

javascript 
// Connect to HeadySystems MCP endpoint
import { MCPClient } from '@heady/mcp-sdk';

const client = new MCPClient({
  endpoint: 'wss://api.headysystems.com/mcp/connect',
  apiKey: process.env.HEADY_API_KEY,
  pipeline: 'FAST_PATH',
  coherenceThreshold: 0.809
});

await client.connect();

// List available tools
const tools = await client.listTools();
console.log(`Available: ${tools.length} MCP tools`);

// Invoke a tool
const result = await client.invoke('heady-search', {
  query: 'sacred geometry topology',
  limit: 10,
  vectorSearch: true
});

Quick Start

Getting Started

Go from zero to fully integrated in under five minutes. Clone, configure, and connect to the Heady ecosystem.

1

Install the Heady CLI

The CLI provides access to all HeadySystems tools, service management, and MCP protocol operations.

bash 
# Install globally via npm
npm install -g @heady/cli

# Verify installation
heady --version
# HeadyCLI v7.1.0 | Pipeline: HCFullPipeline-v7
2

Authenticate & Configure

Connect to the HeadySystems API with your credentials and set up your development environment.

bash 
# Login to HeadySystems
heady auth login

# Initialize project configuration
heady init --pipeline full-path

# Set environment variables
heady config set HEADY_API_KEY "hdy_your_key"
heady config set PIPELINE_MODE "FAST_PATH"
3

Connect to MCP

Establish a connection to the Model Context Protocol for real-time AI-powered development.

bash 
# Start MCP connection
heady mcp connect --mode interactive

# List available tools (45 MCP tools)
heady mcp tools list

# Invoke a tool
heady mcp invoke heady-search \
  --query "architecture topology" \
  --vector-search
4

Deploy Your First Service

Create and deploy a microservice into the Heady ecosystem with built-in observability and CSL gating.

bash 
# Scaffold a new service
heady create service my-service \
  --template microservice \
  --port 3350

# Run locally with hot reload
heady dev --watch

# Deploy to the Heady mesh
heady deploy --target production \
  --region us-east1 \
  --coherence-gate 0.809

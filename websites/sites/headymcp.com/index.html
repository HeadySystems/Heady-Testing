# Set your Heady API key
HEADY_API_KEY=hdy_ent_your_key_here

# Key prefixes by tier:
#   hdy_int_  →  Internal / Developer
#   hdy_pro_  →  Professional
#   hdy_ent_  →  Enterprise

# Optional: MCP Gateway port (default 3500)
MCP_PORT=3500

# Optional: Workspace root for file scoping
HEADY_WORKSPACE=/path/to/your/project
# Health check
curl https://headyio.com/mcp/health

# List all available MCP tools
curl https://headyio.com/mcp/tools \
  -H "Authorization: Bearer $HEADY_API_KEY"

# Call a specific tool
curl -X POST https://headyio.com/mcp/tools/filesystem/call \
  -H "Authorization: Bearer $HEADY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"action":"list","path":"."}'

Configuration
IDE Setup Configs

Drop these into your IDE's MCP settings. Works with Claude Desktop, Cursor, Windsurf, and any MCP-compatible client.

📋 Full Config (headymcp.json) 
{
  "mcpServers": {
    "heady-brain": {
      "url": "https://headyio.com/mcp",
      "apiKey": "${HEADY_API_KEY}",
      "priority": 200,
      "default": true
    },
    "heady-manager": {
      "url": "https://headyio.com/manager",
      "apiKey": "${HEADY_API_KEY}"
    },
    "heady-registry": {
      "url": "https://headyio.com/registry",
      "apiKey": "${HEADY_API_KEY}"
    }
  },
  "modelRouting": {
    "defaultModel": "heady-brain",
    "policy": "heady_only",
    "headyPercentage": 100
  }
}

All HeadyMCP endpoints enforce TLS 1.3 with tier-prefixed API keys.

Tier-prefixed keys: hdy_int_, hdy_pro_, hdy_ent_. Set via HEADY_API_KEY env var.

TLS 1.3 / HTTPS

All HTTP endpoints enforced over HTTPS with modern TLS. No plaintext ever.
